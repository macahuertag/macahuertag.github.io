VOLUMEN DE LUZ

Para esta ocasión el encargo se compuso a partir de una parte análoga y una parte digital. La parte análoga se basó en armar nuevamente una de escena con carton y papel translúcido, de manera que se pudiera mantener en pie. Luego con dos “maquetas”, a modo de planos de luz, se realizaron grabaciones a modo de animación para simular el moviemiento de las siluetas, recogiendo así la mayor cantidad de escalas de tonos grises y negros posibles en movimiento. La parte digital del encargo, se basó en crear un archivo 3D en el programa Fusion360 para luego ser exportado a KeyShot para armar esta misma escena con juegos de luces y de planos para finalmente hacer dos renders animados, de panos diferentes, a modo de video.
